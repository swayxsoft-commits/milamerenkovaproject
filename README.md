# Expense Chart

Консольное приложение на Python для учета личных расходов с возможностью визуализации данных.

## Автор
Меренкова Мила

## Описание
Программа позволяет вести учет расходов, разделяя их на категории и типы (регулярные и необязательные). Реализована валидация пользовательского ввода (проверка формата даты и положительных сумм). Данные автоматически сохраняются в формат JSON. Доступно построение круговой диаграммы расходов с помощью `matplotlib`.

## Как запустить приложение

1. Клонируйте репозиторий или скачайте файлы проекта.
2. Убедитесь, что у вас установлен Python 3.8+.
3. Установите зависимости (требуется библиотека `matplotlib`):
   ```bash
   pip install matplotlib
   ```
4. Запустите приложение:
   ```bash
   python main.py
   ```

## Примеры использования
* **Добавление расхода:** Выберите пункт 1 -> выберите тип -> введите `500` -> введите категорию `Продукты` -> введите дату `2026-06-25`.
* **Построение графика:** Выберите пункт 5 для генерации круговой диаграммы текущих расходов.
